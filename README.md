# Sola Scriptura BR

> A plataforma de estudo bíblico acadêmico mais avançada em português.

## 🚀 Sobre

Sola Scriptura BR é uma plataforma enterprise de estudo bíblico que integra
Inteligência Artificial, exegese, hermenêutica, teologia sistemática,
grego e hebraico bíblicos, história, geografia, arqueologia e muito mais.

## ✨ Funcionalidades

- 📖 **Leitura Bíblica** - Múltiplas traduções lado a lado
- 🔍 **Pesquisa Avançada** - Por palavra, tema, strong, pessoa, cidade
- 📚 **Grego e Hebraico** - Strong, léxico, morfologia completa
- 🧠 **IA Especialista** - RAG + GPT-4 para estudos profundos
- 📐 **Exegese Automática** - Contexto em múltiplos níveis
- 📖 **Hermenêutica** - Gêneros literários e aplicação
- ⛪ **Teologia** - Doutrinas e tradições teológicas
- 🏛️ **História** - Contexto político, religioso e cultural
- 🗺️ **Geografia** - Mapas interativos com localizações
- 🏺 **Arqueologia** - Artefatos, manuscritos e escavações
- 👤 **Personagens** - Biografias e genealogias
- 📅 **Cronologia** - Linha do tempo completa
- 🔗 **Referências Cruzadas** - Grafo de conexões bíblicas

## 🛠️ Stack

| Camada | Tecnologia |
|--------|-----------|
| Frontend | Next.js, TypeScript, TailwindCSS, ShadCN |
| Mobile | Flutter |
| Backend | NestJS |
| Banco | PostgreSQL + pgvector + Elasticsearch |
| Cache | Redis |
| Mensageria | RabbitMQ |
| IA | OpenAI GPT-4 + RAG |
| Infra | Docker + Kubernetes |

## 🏁 Começando

```bash
git clone https://github.com/sola-scriptura/plataforma.git
cd plataforma
npm install
npm run docker:up
npm run dev
```
